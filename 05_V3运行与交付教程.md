# 05_V3运行与交付教程

## 1. 自己运行

进入：

```text
D:\Yiqian_Yang _Homework_c++\project\fullstack_v3
```

双击：

```text
run_backend.bat
```

浏览器会打开：

```text
http://localhost:8090
```

第一次使用需要注册账号。

## 2. VS Code 编译

打开 `fullstack_v3` 文件夹。

按：

```text
Ctrl + Shift + B
```

会使用 `.vscode/tasks.json` 中的命令编译。

## 3. 命令行编译

```powershell
cd "D:\Yiqian_Yang _Homework_c++\project\fullstack_v3\backend"
g++ server.cpp -std=c++17 -finput-charset=UTF-8 -fexec-charset=UTF-8 -static -static-libgcc -static-libstdc++ -o server.exe -lws2_32
```

```text
D:\Yiqian_Yang _Homework_c++\project\fullstack_v3_release_给同学运行
```

## 5. 答辩说明

可以这样说：

V3 在 V2 前端 + C++ 后端结构基础上增加了用户注册和登录功能。后端新增 User 结构体和用户文件，前端新增登录注册页面。任务结构体新增 username 字段，所有任务查询、统计、提醒、打卡和删除都根据 username 过滤，实现了不同用户之间的数据隔离。
