# V3 测试报告

## 一、测试目标

验证 V3 新增用户注册、登录和用户数据隔离功能，同时确认 V2 原有任务管理、提醒、打卡、统计功能仍然正常。

## 二、测试环境

- Windows
- MinGW g++
- Chrome / Edge
- 后端端口：8090
- 后端文件：backend/server.exe

## 三、测试范围

1. 注册功能
2. 登录功能
3. 退出登录
4. 用户数据隔离
5. 当前用户任务添加
6. 当前用户任务打卡
7. 当前用户任务删除
8. 当前用户提醒
9. 当前用户统计
10. 用户数据文件保存
11. 任务数据文件保存

## 五、测试结论

V3 版本实现了注册、登录和按用户隔离任务数据。不同用户登录后只能看到自己的任务、提醒、打卡记录和统计数据。系统满足课程项目对学习养成计划软件的主要功能要求。

## 六、遗留问题

1. 密码当前为明文保存，适合课程演示，不适合真实上线。
2. 没有使用数据库，仍然使用文本文件保存。
3. 没有真正的 Session 或 Cookie，前端使用 localStorage 保存当前用户信息。
4. 后端 HTTP 解析适合本地演示，不适合高并发生产环境。
